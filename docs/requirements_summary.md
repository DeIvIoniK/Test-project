# Анкета / требования к Telegram ИИ‑помощнику для поддержки в выздоровлении

Источник: `/opt/data/aa_support_bot_requirements.md`.

Полная анкета используется как продуктовый документ проекта. Основные решения уже отражены в коде MVP:

- публичный бот: Python + aiogram + FastAPI;
- стиль: коротко, на «ты», без давления и стыда;
- безопасность: не врач, не психолог, не экстренная служба, не официальный представитель АА;
- кризис: коротко, один вопрос за раз, фокус на безопасности;
- диалоги: TTL 30 минут;
- меню: поддержка, SOS, связь с человеком, новичок, группы, литература, аудио, дни трезвости, помочь другому, поддержать проект, настройки;
- админ: только владелец проекта.
